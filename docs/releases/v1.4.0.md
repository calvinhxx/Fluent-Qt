Fluent-Qt 1.4.0 makes the Gallery easier to read and adapt across supported desktops, with portable design assets, configurable display scaling, richer foundation references, and a broad component polish pass.

## Portable typography and iconography

- FluentQt now bundles redistributable typography and Fluent System Icons assets, removing reliance on platform-installed Segoe fonts while keeping the intended Fluent visual language.
- Icons are resolved through the bundled catalog and rendered at the target display density, improving sharpness and consistency across Windows, macOS, Linux, and custom scale factors.

## Richer Gallery documentation

- Foundation now has focused pages for QML+, typography, color, iconography, geometry, and spacing, including a searchable browser for the complete bundled icon catalog.
- Gallery samples, captions, code snippets, and responsive layouts have been aligned so the documented behavior matches the live preview more closely.

## Display scaling and window behavior

- FluentQt follows the operating-system scale, keeps Gallery window placement in logical coordinates, and validates custom surfaces and raster density from 110% through 300% without an additional Gallery multiplier.
- macOS now relaunches the complete application bundle after a scale change and restores it to the foreground. Windows custom-scale resize hit testing and active-title-bar feedback are also more reliable.
- Dialogs and combo-box popups fit their content more consistently across supported platforms and Qt versions.

## Component and interaction fixes

- Themed input, rating, and split-button states once again provide the expected visual feedback.
- Drawer edge gestures no longer activate from hidden hosts, and TreeView indicator animations no longer leave stale paint trails.
- Collection layouts, typography, icon metrics, and source expansion behavior have been tightened throughout the Gallery.

## Distribution

- Runtime packages and the source archive now include the applicable third-party license and dependency notices alongside the project license.

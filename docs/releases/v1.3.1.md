Fluent-Qt 1.3.1 makes the component library easier to integrate and keeps the Gallery packages focused on the Qt Widgets features they actually use.

## Library integration

- The reusable FluentQt library now depends only on Qt Widgets. spdlog/fmt remains with Gallery diagnostics, and GTest remains with the test suite.
- `FetchContent`, source-subproject, and installed-package integration are documented and continuously verified with Qt 5.15 and Qt 6.2.
- Releases now include `FluentQt-1.3.1-source.zip`, a library-only source archive without Gallery, tests, or application logging support.

## Smaller Gallery packages

- Windows and macOS packages no longer include unused translations, QML/Qt Quick, or OpenGL fallback payloads.
- The standard package matrix now uses the intended Qt toolchain for every desktop platform and architecture.
